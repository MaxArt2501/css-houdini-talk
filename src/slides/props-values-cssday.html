Properties and values

#1: the cascade

.box {
  width: 20em;
  width: red;
}
width: red gets ignored

What happens now?

.box {
  --box-width: 20em;
  --box-width: red;
  width: var(--box-width);
}
It's --box-width: 20em that gets ignored!
