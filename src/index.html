Internet Explorer

introduced 2 ways JavaScript could extend CSS:

  • 👉 expressions;
  • 👉 behaviors.

IE's CSS expressions

.indicator {
  width: 100%;
  color: expression(this.clientWidth > 400 ? 'red' : 'blue');
}
